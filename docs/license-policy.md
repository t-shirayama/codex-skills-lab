# License Policy

このリポジトリでは、Agent Skillsを安全に共有・改善するため、次のライセンス方針に従います。

## 基本方針

- 他人のSkillを丸ごとコピーしない。
- 参考にする場合は、構成や考え方を理解したうえで自分の言葉で書き直す。
- コピー・改変する場合は、元ライセンスに従う。
- ライセンス不明のものは取り込まない。
- 必要な出典は `NOTICE.md` または該当Skill内に書く。
- このリポジトリ自体はMITライセンスを想定する。

## 参考にする場合

1. 参考元のURL、リポジトリ名、作者、ライセンスを確認する。
2. ライセンスが再利用や改変を許可しているか確認する。
3. 本文をそのまま写さず、目的に合わせて書き直す。
4. 必要な帰属表示を `NOTICE.md` または該当Skill内に記載する。
5. ライセンス条件に不明点がある場合は取り込まない。

## コピー・改変する場合

1. コピーや改変が許可されているライセンスか確認する。
2. 元ライセンスが求める著作権表示、ライセンス表示、変更表示を残す。
3. `NOTICE.md` に出典とライセンス情報を書く。
4. 取り込んだ範囲が分かるように記録する。

## 取り込まないもの

- ライセンスが不明なSkillやドキュメント
- 秘密情報、個人情報、社内情報を含む内容
- 外部サービスの利用規約に反する可能性がある内容
- 出典を確認できない本文のコピー
