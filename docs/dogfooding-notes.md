# Dogfooding Notes

このドキュメントは、`docs/skill-usage-examples.md` の依頼例を実際にCodexで試した結果を記録するための手動検証ログです。

## 記録方法

1. `docs/skill-usage-examples.md` から依頼例を1つ選ぶ。
2. Codexに依頼し、どのSkillが使われたかを確認する。
3. 期待したSkillと実際に使われたSkill、結果、改善点を表に記録する。
4. 改善が必要な場合は、該当Skillの `description`、使う場面、使わない場面、出力形式を見直す。
